#Database Module: Digital OSBB

Цей модуль містить архітектуру бази даних для системи управління ОСББ. Структура розроблена з урахуванням інтеграції через **Дія.Підпис** та автоматизації паперової роботи (PDF-звіти).

##Технологічний стек
* **RDBMS:** MySQL 8.0
* **Charset:** `utf8mb4` (для коректного відображення української мови)
* **Tooling:** MySQL Workbench / Command Line

##Основні таблиці
- **`residents`**: Реєстр мешканців. Ключове поле — `tax_id` (РНОКПП), яке використовується для верифікації через API Дії.
- **`polls` & `poll_votes`**: Модуль для легітимних онлайн-голосувань. Передбачено поле `diia_signature_token` для зберігання криптографічного доказу підпису.
- **`announcements`**: Система сповіщень для заміни Viber-чатів.
- **`documents`**: Метадані для PDF-документів (статути, протоколи, рахунки).

##Як підключити та розгорнути

Щоб підключити цю базу до вашого локального середовища розробки, виконайте наступні кроки:

### 1. Імпорт схеми
1. Відкрийте **MySQL Workbench**.
2. Створіть нове підключення (Connection) до вашого локального сервера.
3. Відкрийте файл `database/init_db.sql` з цього репозиторію.
4. Натисніть іконку ⚡ (**Execute**), щоб створити базу `digital_osbb` та всі таблиці.

### 2. Налаштування підключення (Connection String)
Для підключення Backend-частини (Node.js/Python/Java) використовуйте наступні параметри:
* **Host:** `localhost` (або IP вашого сервера)
* **Port:** `3306`
* **User:** `root`
* **Database:** `digital_osbb`
* **Password:** *Ваш локальний пароль від MySQL*
