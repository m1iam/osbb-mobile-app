# Definition of Done

Завдання вважається виконаним, якщо:

- виконані всі вимоги до задачі;
- код або результат перевірений;
- немає критичних помилок;
- зміни додані до GitHub;
- документація оновлена (якщо потрібно);
- завдання позначене як Done в Jira.

## Документаційний критерій
Кожне завдання повинно бути відображене в документації, якщо воно впливає на систему.
