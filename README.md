# Цифрове ОСББ

## Опис проєкту
Цифрове ОСББ — це мобільний застосунок для автоматизації управління багатоквартирним будинком, що забезпечує зручну взаємодію між мешканцями та адміністрацією ОСББ.

## Мета проєкту
Метою проєкту є створення інформаційної системи, яка дозволить:
- організувати електронну взаємодію між мешканцями та адміністрацією;
- автоматизувати облік платежів;
- впровадити електронний документообіг;
- реалізувати систему подачі заявок;
- проводити електронні голосування;
- швидко інформувати мешканців про важливі події.

## Основний функціонал
### Для мешканця:
- авторизація за номером телефону;
- перегляд оголошень;
- подання заявок про проблеми;
- участь у голосуваннях;
- перегляд документів;
- онлайн-оплата внесків;
- отримання електронних квитанцій.

### Для адміністратора:
- керування базою мешканців;
- обробка заявок;
- публікація оголошень;
- масові розсилки;
- фінансова звітність;
- контроль підписання документів.

## Технології
- React Native
- Node.js
- MySQL
- SQLite

## Quick Start
1. Клонувати репозиторій:
   `git clone https://github.com/m1iam/osbb-mobile-app.git`

2. Перейти до папки проєкту:
   `cd osbb-mobile-app`

3. Встановити залежності:
   `npm install`

4. Запустити проєкт:
   `npm start`
